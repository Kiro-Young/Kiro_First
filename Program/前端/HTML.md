# HTML学习笔记

* HTML(Hyper Text Markup Language),是用来描述网页的一种语言；不是编程语言，而是标记语言（markup lanhuage）；标记语言是一套标记标签（markup tag）
* “超文本”含义：可加多媒体，超“文本”的限制；可进行文件跳跃，与世界各地主机文件相连，即“超级链接文本”
* 网页HTML文件由前端写出，再由浏览器打开即是网页
